Pause Syntax Help

How to use pause tags:

Insert pause tags directly in your text to add silence:

<pause 500> - pause for 500 milliseconds

<pause 1.5s> - pause for 1.5 seconds

<pause> - default pause (500ms)

Speed Control Tags:

Control speech speed within your text:

<slow>text</slow> - speaks slower

<fast>text</fast> - speaks faster

Example:

Hello<pause 800>how are you? <slow>I'm doing great</slow>!<pause 1s>Thanks for asking <fast>have a nice day</fast>.

Notes:

  • Pause tags are removed from the final audio
  • You can use milliseconds (default) or seconds (add 's' suffix)
  • Pauses work with all languages and voices
  • Speed tags (<slow> and <fast>) work locally in any part of text
  • The server needs FFmpeg installed for pause support
